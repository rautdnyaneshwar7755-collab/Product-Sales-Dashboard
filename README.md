

# Product Sales Analysis Dashboard

![Dashboard](https://github.com/rautdnyaneshwar7755-collab/Product-Sales-Dashboard/blob/main/Screenshot%202025-12-08%20171442.png)

## Overview
This dashboard provides key insights into product sales, profit, customer behavior, payment modes, and regional performance.

## Key Metrics
- 438K – Total Sales Amount  
- 37K – Total Profit  
- 5615 – Quantity Sold  
- 121K – Average Order Value  

## Dashboard Sections
### 1. Top States
Shows top-performing states like Maharashtra, Madhya Pradesh, Uttar Pradesh, etc.

### 2. Quantity by Category
- Clothing – 63%
- Electronics – 21%
- Furniture – 17%

### 3. Profit–Loss by Month
Visualizes monthly profit/loss trends for strategic planning.

### 4. Top Customers
Lists customers contributing most to revenue.

### 5. Quantity by Payment Mode
COD (44%), UPI (21%), Debit Card (13%), Credit Card (12%), EMI (10%).

### 6. Profit by Sub-Category
Printers, Bookcases, Saree, Accessories, Tables.
